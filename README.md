# AI-Powered Drink & Recipe Finder 🍹

A high-performance React application designed to discover beverages and recipes through a seamless, type-safe interface. This project features **AI-driven search optimization** to enhance user experience.

## 🛠️ Tech Stack
* **Core:** React 19 + TypeScript + Vite.
* **State Management:** **Zustand** (Slice Pattern for scalability).
* **Data Validation:** **Zod** (ensuring type safety for API responses).
* **Networking:** Axios + React Router Dom.
* **Styling:** Tailwind CSS.

## 🚀 Key Features
* **AI Search Integration:** Leverages an AI model to streamline recipe discovery.
* **Complex State:** Managed via Zustand slices for independent and clean logic.
* **Ingredient Filtering:** Real-time data fetching from professional API endpoints.
* **Persistent Favorites:** Global state management to save user preferences.

## ⚙️ Setup
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Create a `.env` file based on the environment variables required by the Drink API and the AI Model.
4. Run the development server: `npm run dev`.
